# Vox-PDF

Convert PDF documents into natural-sounding speech using Flask and Microsoft Edge TTS.

## Features

- Upload PDF files
- Extract text from all pages
- Convert text to speech
- Automatically download the generated MP3
- Clean and responsive interface

## Tech Stack

- Flask
- JavaScript
- HTML/CSS
- PyPDF
- Edge TTS
